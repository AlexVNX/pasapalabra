# Decks oficiales

- `oficiales.es.json`: contenido oficial en español.

Formato:
- `title`, `description`
- `cards[]`: cada tarjeta con `id`, `question`, `answer` y opcionales `hint`, `explanation`, `tags`.

Nota:
- Para que Pasapalabra funcione por letras, intenta que la **respuesta** empiece con letras variadas (incluye Ñ si quieres).
